> Built with Yeoman


##### install Node.js
reference - http://thechangelog.com/install-node-js-with-homebrew-on-os-x/

##### install Yeoman
```
npm install -g yo
```

## Building the app
* cd into cloned directory
* install dependencies with 
```
bower install
```

#### for dev run 

```
gulp watch
```

#### to build run 

```
gulp
```
* compiled files are placed in the dist directory
